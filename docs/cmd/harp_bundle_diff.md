## harp bundle diff

Display container differences

```
harp bundle diff [flags]
```

### Options

```
  -h, --help         help for diff
      --new string   Container path ('-' for stdin or filename)
      --old string   Container path ('-' for stdin or filename)
      --patch        Output as a bundle patch
```
