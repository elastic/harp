## harp from oplog

Convert a JSON oplog to a secret-container

```
harp from oplog [flags]
```

### Options

```
  -h, --help         help for oplog
      --in string    JSON OpLog object ('-' for stdin or filename) (default "-")
      --out string   Container output ('-' for stdout or filename) (default "-")
```
