# Introduction

`harp` defines and uses :

## Secret Storage

* `Bundle` to describe how to store data and metadata;

## Secret Operations

* `BundleTemplate` to describe how to generate a `Bundle` from requirements;
* `BundlePatch` to describe how to update a `Bundle`.
